CASE STUDY: AUTOMOTIVE

UX Design · Automotive · In-Car Media

One Radio.
Every Road.

Designing a global automotive platform—balancing strict legal requirements and cognitive load across multiple global markets.

Overview

I joined as lead UX designer to define the conditional logic for edge cases and completed key features: a legally mandated Public Value indicator, a redesigned Manual Tuner, and a market-adaptive Settings structure.

Market Specific Logic

Germany: Mandatory Public Value (PV) indicators.

USA/Japan: Frequency-first display and separate AM/FM lists.

Global: Unified DAB/FM list entries.
